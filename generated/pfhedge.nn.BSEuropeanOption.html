Black-Scholes formula for a European option.

Parameters

  • derivative (pfhedge.instruments.EuropeanOption, optional) â The derivative to get the Black-Scholes formula.

  • call (bool, default=True) â Specifies whether the option is call or put.

  • strike (float, default=1.0) â The strike price of the option.

Shape:

  • Input : \((N, *, 3)\), where \(*\) means any number of additional dimensions. See features() for input features.

  • Output: \((N, *, 1)\). Delta of the derivative. All but the last dimension are the same shape as the input.

See also

  • pfhedge.nn.BlackScholes: Initialize Black-Scholes formula module from a derivative.

Examples

The forward method returns delta of the derivative.

>>> from pfhedge.nn import BSEuropeanOption
>>> m = BSEuropeanOption()
>>> m.features()
['log_moneyness', 'expiry_time', 'volatility']
>>> input = torch.tensor([
...     [-0.01, 0.1, 0.2],
...     [ 0.00, 0.1, 0.2],
...     [ 0.01, 0.1, 0.2]])
>>> m(input)
tensor([[0.4497],
        [0.5126],
        [0.5752]])

One can instantiate it using a pfhedge.instruments.EuropeanOption.

>>> from pfhedge.instruments import BrownianStock
>>> from pfhedge.instruments import EuropeanOption
>>> deriv = EuropeanOption(BrownianStock(), call=False)
>>> m = BSEuropeanOption(deriv)
>>> m
BSEuropeanOption(call=False)
